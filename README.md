<img src="images/gmail.png" width="10%" />


## Overview
Scripts that helps you to remember forgotten password to **Gmail**


## How to use
Put all emails and possible passwords in **list.txt** in format : **email:password** .
Example of **list.txt**:
```
sample@gmail.com:password123

```


## Usage example
```bash
python2 main.py

```
